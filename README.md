## Overview

RNK Mystix is a Foundry Virtual Tabletop module for Pathfinder 2nd Edition that enables Game Masters to assign both Hero Points and Mystic Points to actors simultaneously. When rolls are made, both point types contribute bonuses that are totaled together with the base roll result.

## Features

- **GM Point Assignment Hub**: Dropdown/search interface for filtering and assigning points to PCs, NPCs, and Mystics
- **Synchronized Roll Integration**: Hero Points and Mystic Points bonuses automatically added to all roll types (attacks, skills, saves)
- **Visible Point Totals**: Point assignments display on character sheets for all players to see
- **Flexible Configuration**: Set maximum points, defaults, and toggle options through settings
- **Per-Actor & Global Reset**: Reset points individually or globally from the GM hub
- **Clean Design**: Professional, legible interface with gold (Hero Points) and purple (Mystic Points) color coding

## Installation

1. In Foundry VTT, go to **Add-on Modules**
2. Click **Install Module**
3. Search for "RNK Mystix" or paste the manifest URL
4. Click **Install**
5. Activate the module in your world

## Usage

### Opening the GM Hub

1. Log in as GM
2. Look for the "RNK Mystix" button in the toolbar
3. Click to open the Point Assignment Hub

### Assigning Points

1. Use the dropdown or search to find an actor
2. Actors are organized by type (PCs, NPCs, Mystics)
3. Enter Hero Point and Mystic Point values
4. Click **Assign** to apply

### Rolls

- When an actor with assigned points makes a roll, both Hero and Mystic point bonuses automatically add to the total
- The roll message will show the breakdown of the combined bonus

### Settings

Customize the module behavior through the settings panel:
- Maximum Hero Points per actor
- Maximum Mystic Points per actor
- Default point values
- Toggle for negative points (debt system)
