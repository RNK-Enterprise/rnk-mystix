## Overview

RNK™ Mystix gives Game Masters a focused way to manage Hero Points and Mystic Points for Pathfinder 2e actors. The module provides a GM hub for point assignment, sheet display for tracked totals, and chat-card reroll spending for supported roll types.

## Features

- GM hub for assigning and resetting Hero Points and Mystic Points
- actor sheet display with optional player self-adjustment
- chat-card spending buttons for Hero and Mystic rerolls
- configurable point caps, defaults, negative values, rest behavior, and reroll scope
- synchronized storage using actor flags on world actors

## Installation

This is a premium Patreon-gated RNK™ module.

1. Open **Add-on Modules** in Foundry VTT.
2. Choose **Install Module**.
3. Use this manifest URL:

```text
https://github.com/RNK-Enterprise/rnk-mystix/releases/latest/download/module.json
```

4. Install the module.
5. Enable it in your world configuration.

## Usage

### GM hub

The GM scene-control button opens the point assignment hub. From there you can:

- search actors by name
- assign Hero and Mystic point values
- clear an actor’s stored values
- reset an actor back to the configured defaults
- reset all stored values across the world

### Sheet display

Actor sheets show current Hero and Mystic totals. GMs can always adjust values from the sheet display, and players can do the same when the self-assignment setting is enabled.

### Chat spending

When supported roll cards render in chat, actors with available points receive spending buttons for Hero and Mystic rerolls.

## Settings
